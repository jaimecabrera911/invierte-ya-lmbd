🏦 Invierte Ya - API de Fondos

Documentación interactiva de la API para gestión de fondos de inversión

📋 Información de la API

Versión: 1.0.0

Servidor de Producción: https://jwnazw2b41.execute-api.us-east-1.amazonaws.com/Prod

Descripción: API para gestión de fondos de inversión FPV (Fondos de Pensiones Voluntarias) y FIC (Fondos de Inversión Colectiva)

🏥 Health & Status

GET/ - Endpoint raíz
GET/health - Verificación de salud

👥 Gestión de Usuarios

POST/users - Crear usuario
GET/users/{user_id} - Obtener usuario

💰 Fondos de Inversión

GET/funds - Listar fondos
POST/funds/subscribe - Suscribirse
POST/funds/cancel - Cancelar suscripción

📊 Transacciones & Suscripciones

GET/users/{user_id}/transactions - Historial
GET/users/{user_id}/subscriptions - Suscripciones activas

⚙️ Administración

POST/init-funds - Inicializar fondos
